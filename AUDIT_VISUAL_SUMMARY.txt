⚠️  CRITICAL WARNINGS
═════════════════════════════════════════════════════════════════════════════════

    1. DO NOT add z-index classes to ResponsiveDialog
       ❌ WRONG: <ResponsiveDialog className="z-[999]" />
       ✅ RIGHT: <ResponsiveDialog />  (LayerManager handles it)

    2. DO NOT implement custom escape key handling
       ❌ WRONG: useEffect(() => { window.addEventListener('keydown', ...) })
       ✅ RIGHT: Let LayerManager handle it automatically

    3. DO NOT conditionally render Dialog vs Sheet
       ❌ WRONG: {isMobile ? <Sheet /> : <Dialog />}
       ✅ RIGHT: <ResponsiveDialog />  (auto-switches)

    4. DO NOT use raw Dialog for new components
       ❌ WRONG: import { Dialog } from "@/components/ui/dialog"
       ✅ RIGHT: import { ResponsiveDialog } from "@/components/canvas-hub"

    5. DO TEST on both mobile AND desktop before marking complete
       Minimum: Resize viewport from 320px to 1920px
