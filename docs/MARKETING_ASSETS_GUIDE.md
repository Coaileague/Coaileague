## 🎨 Visual Style Guidelines

### Color Scheme for Screenshots
- **Primary**: Indigo/Blue (#6366f1)
- **Success**: Green (#10b981)
- **Warning**: Orange/Amber
- **Error**: Red
- **Background**: Dark theme (default)

### Typography
- **Headings**: Bold, clear hierarchy
- **Body**: Easy to read at any size
- **Monospace**: For numbers, stats, data

### What to Highlight
✅ **DO Show:**
- Real data (from demo workspace)
- Smooth animations
- Professional UI elements
- CAD-style interface
- Live indicators (clock, status)
- Interactive elements (buttons, cards)

❌ **DON'T Show:**
- Empty states
- Loading spinners
- Error messages (unless showing error handling)
- Incomplete features
- Browser dev tools (unless educational)

## 🚀 Quick Capture Instructions

### Method 1: Browser Screenshot (Built-in)
1. Press `F12` (open DevTools)
2. Press `Ctrl+Shift+P` (Command Palette)
3. Type "screenshot"
4. Select "Capture full size screenshot"
5. Image saves to Downloads

### Method 2: Browser Extension
1. Install "Awesome Screenshot"
2. Click extension icon
3. Select "Capture Visible Part"
4. Annotate if needed
5. Download PNG

### Method 3: Operating System
**Windows:**
- `Win+Shift+S` → Select area

**Mac:**
- `Cmd+Shift+4` → Select area
- `Cmd+Shift+4 → Space` → Capture window
