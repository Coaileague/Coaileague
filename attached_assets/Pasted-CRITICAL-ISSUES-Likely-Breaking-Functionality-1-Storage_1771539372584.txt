Routes call storage methods that may not be defined in storage.ts. Examples include acknowledgeNotification, clearNotification, closeChatConversation, createClient, and others. These would cause runtime errors when those code paths are hit.

2. Frontend-Backend Endpoint Gap
The frontend references ~495 API endpoints, but only ~238 are captured as explicit route definitions. Even accounting for inline routers, the delta suggests some API calls from the UI may be hitting routes that were moved or removed during refactoring. This means some UI features could silently fail.

3. Unregistered Route Files
Several route files exist but aren't registered in the main routes.ts: chat management, dispatch, email attachments, security audit, and others. Those entire API surfaces are unreachable.
