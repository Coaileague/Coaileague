Totally fixable. The trick is progressive disclosure on small screens: show the essentials first, fold the rest behind taps. Below is a drop-in toolkit you can use on any page to make your AutoForce site compact on mobile while keeping all info accessible.

1) Drop-in CSS (compact mobile layout)

Paste this into your <head> inside a <style> tag or your CSS file.

/* ===== Base responsive + density controls ===== */
* { box-sizing: border-box }
html,body { margin:0; padding:0; width:100%; overflow-x:hidden; }
img,video,iframe { max-width:100%; height:auto; display:block; }

:root{
  --maxw: 1200px;
  --gap: clamp(12px, 2.5vw, 24px);
  --pad: clamp(12px, 3vw, 24px);
  --radius: 12px;
  --txt: clamp(15px, 1.6vw, 17px);
  --h2: clamp(20px, 3.2vw, 28px);
  --muted:#6e727a;
}

.container { max-width: var(--maxw); margin: 0 auto; padding: var(--pad); }
.row { display:flex; flex-wrap:wrap; gap: var(--gap); align-items:flex-start; }
.col { flex:1 1 320px; min-width: 280px; }

/* Compact spacing on mobile */
@media (max-width: 768px){
  .container { padding: 12px }
  .row { gap: 12px }
  .stack { display:grid; gap: 10px }             /* generic stacked blocks */
  .tight > * { margin-block: 6px }               /* reduce vertical gaps */
  .hide-desktop { display: initial }
  .hide-mobile { display: none !important }
}

/* Desktop-only / Mobile-only helpers */
.hide-desktop { display:none }                   /* default hidden; shown on mobile via media above */
.hide-mobile { display: initial }                /* default shown; hidden on mobile via media above */

/* ===== Progressive disclosure patterns ===== */

/* A) Line clamp for long text on mobile with 'Show more' */
.clamp-3 { display:-webkit-box; -webkit-box-orient:vertical; overflow:hidden; -webkit-line-clamp:3; }
@media (min-width: 769px){ .clamp-3 { -webkit-line-clamp:unset; } }

/* B) Details/summary accordion (native, no JS) */
details { border:1px solid #ececec; border-radius: var(--radius); padding:10px 12px; background:#fff; }
details + details { margin-top: 10px }
summary { cursor:pointer; font-weight:600; }

/* C) Cards → horizontal carousel on mobile */
.card-grid { display:grid; grid-template-columns: repeat(3, minmax(0,1fr)); gap: var(--gap);}
@media (max-width: 768px){
  .card-grid { display:flex; gap:12px; overflow-x:auto; scroll-snap-type:x mandatory; padding-bottom:8px; }
  .card-grid .card { min-width: 80%; scroll-snap-align: start; }
}
.card { background:#fff; border:1px solid #ececec; border-radius: var(--radius); padding:16px; }

/* D) Tables → stacked lists on mobile */
.table { width:100%; border-collapse: collapse; }
.table th, .table td { border-bottom:1px solid #ececec; padding:10px; text-align:left; }
@media (max-width: 768px){
  .table, .table thead, .table tbody, .table tr, .table th, .table td { display:block; }
  .table thead { display:none; }
  .table tr { background:#fff; border:1px solid #ececec; border-radius: var(--radius); padding:8px; margin-bottom:10px; }
  .table td { border:0; padding:6px 8px; }
  .table td[data-label]::before { content: attr(data-label) ": "; font-weight:600; color:#444; }
}

/* E) Section index: sticky jump menu (mobile) */
.section-nav { position: sticky; top: 0; z-index: 30; background:#fff; border-bottom:1px solid #ececec; }
.section-nav .chips { display:flex; gap:10px; overflow-x:auto; padding:8px var(--pad); }
.chip { white-space:nowrap; border:1px solid #e3e3e3; border-radius:999px; padding:6px 10px; font-size:14px; }

/* F) Sticky bottom action bar (mobile CTA) */
@media (max-width: 768px){
  .cta-bar { position: sticky; bottom: 0; background:#101113; color:#fff; padding:10px var(--pad); display:flex; gap:10px; align-items:center; justify-content:space-between; }
  .cta-bar .btn { background:#f7b500; color:#101113; border-radius:10px; padding:10px 14px; border:0; }
}

/* G) Secondary blocks collapsible on mobile */
.mobile-collapse { }
@media (max-width: 768px){
  .mobile-collapse .collapse-body { display:none; }
  .mobile-collapse[aria-expanded="true"] .collapse-body { display:block; }
}

3) Content strategy (so mobile isn’t “infinite scroll”)

Prioritize: put P1 info (what it is, why it helps, one CTA) above the fold; push P2 into accordions.

Summarize: show 2–3 lines with .clamp-3, include “Show more” toggles.

Group: convert large grids into a carousel on mobile (users swipe).

Jump links: Add the sticky section nav so users hop between sections.

Sticky CTA: bottom action bar on mobile to avoid scrolling back up.

Shorten spacing: .tight and smaller gap on mobile reduce empty space.
