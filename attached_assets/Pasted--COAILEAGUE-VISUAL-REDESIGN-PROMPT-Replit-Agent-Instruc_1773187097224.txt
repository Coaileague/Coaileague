### Color Palette (CSS Variables)
```css
:root {
  /* Brand */
  --color-navy:        #0B1629;   /* primary dark background */
  --color-navy-mid:    #102040;   /* card backgrounds */
  --color-navy-light:  #1A3058;   /* elevated surfaces, hover states */
  --color-border:      #1E3A5F;   /* subtle borders */

  /* Accent */
  --color-gold:        #C9952A;   /* primary accent — CoAIleague brand */
  --color-gold-light:  #E8B84B;   /* hover gold */
  --color-gold-glow:   rgba(201,149,42,0.15); /* ambient glow */

  /* Status */
  --color-success:     #22C55E;
  --color-warning:     #F59E0B;
  --color-danger:      #EF4444;
  --color-info:        #3B82F6;

  /* Text */
  --color-text-primary:   #F0F4FF;
  --color-text-secondary: #8BA3C7;
  --color-text-muted:     #4A6A96;

  /* Trinity */
  --color-trinity:     #7C3AED;   /* purple — Trinity AI actions only */
  --color-trinity-glow: rgba(124,58,237,0.2);
}
```

### Typography
- **Display / Page Titles**: `font-family: 'Rajdhani', sans-serif` — weight 600–700. Military-precision feel, modern, readable. Import from Google Fonts.
- **Body / Labels / Data**: `font-family: 'DM Sans', sans-serif` — weight 400–500. Clean, professional, not Inter.
- **Monospace / IDs / Codes**: `font-family: 'JetBrains Mono', monospace` — for doc numbers, unit IDs, timestamps.

### Universal Component Specs

#### `<PageHeader>` — Used on EVERY page
```
Layout: page title (Rajdhani, 28px, text-primary) + subtitle (DM Sans, 14px, text-secondary, gold color)
Optional: right-side action button slot
Bottom: 1px border in --color-border
Background: transparent (inherits page bg)
Mobile: title 22px, subtitle wraps naturally
```

#### `<StatCard>` — Used on EVERY dashboard/overview page
```
Background: --color-navy-mid
Border: 1px solid --color-border
Border-radius: 12px
Padding: 20px 24px
Number: Rajdhani 36px bold, colored by context (gold=neutral, green=good, red=alert, amber=warning)
Label: DM Sans 12px uppercase tracking-widest text-muted
Hover: border transitions to --color-gold, subtle box-shadow glow
Mobile: grid 2-col on mobile, 4-col on desktop
```

#### `<TabBar>` — Used on all tabbed pages
```
Background: --color-navy-mid
Border-radius: 10px
Padding: 4px
Each tab: DM Sans 13px, text-muted when inactive
Active tab: bg --color-navy-light, text-primary, left border 2px --color-gold
Hover: text transitions to text-primary
Mobile: horizontally scrollable, no wrap
```

#### `<EmptyState>` — Used on all empty list pages
```
Icon: outlined, --color-gold opacity-40, 48px
Title: Rajdhani 20px text-secondary
Subtitle: DM Sans 14px text-muted
CTA button (if applicable): gold outline button
Center aligned, generous vertical padding (py-16)
```

#### `<DataRow>` / `<ListItem>` — Used in all list views
```
Background: --color-navy-mid
Border-bottom: 1px solid --color-border
Padding: 14px 20px
Hover: bg transitions to --color-navy-light
Status badges: pill shape, 11px DM Sans uppercase, colored bg with matching text
Mobile: stack secondary info below primary title
```

#### `<SectionCard>` — Used for content panels
```
Background: --color-navy-mid
Border: 1px solid --color-border
Border-radius: 12px
Header: Rajdhani 16px uppercase tracking-wide text-secondary, border-bottom --color-border
Body padding: 20px
```

#### Page background
```
All pages: bg-[#0B1629] (--color-navy)
Subtle noise texture overlay: 3% opacity SVG noise pattern for depth
```

#### Buttons
```
Primary: bg --color-gold, text black, font-semibold, rounded-lg, hover bg --color-gold-light, shadow-md
Secondary/Outline: border --color-gold text --color-gold bg transparent, hover bg --color-gold-glow
Danger: bg --color-danger/10 border --color-danger text --color-danger
Trinity action: bg --color-trinity/20 border --color-trinity text white, glow on hover
```
