- Dark background with cards that have no visual personality

- Buttons that look like every other SaaS app

- Skeleton loaders that match the shape of the content loading

- Color used intentionally — cyan for primary, green for success, amber for warning, red for danger
- Every page has one dominant visual element that anchors the layout

## SECTION 2 — GLOBAL ANIMATION SYSTEM

Add this to the global stylesheet. Apply immediately everywhere.

```css
/* Page entry */
@keyframes fadeUp {
  from { opacity: 0; transform: translateY(12px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* Card/element entry */
@keyframes slideIn {
  from { opacity: 0; transform: translateY(8px); }
  to   { opacity: 1; transform: translateY(0); }
}

/* Shimmer skeleton */
@keyframes shimmer {
  0%   { background-position: -200% 0; }
  100% { background-position:  200% 0; }
}

/* Pulse glow for live indicators */
@keyframes pulseGlow {
  0%, 100% { box-shadow: 0 0 0 0 rgba(0,212,255,0.4); }
  50%       { box-shadow: 0 0 0 6px rgba(0,212,255,0); }
}

/* Success pop */
@keyframes successPop {
  0%   { transform: scale(0.8); opacity: 0; }
  70%  { transform: scale(1.1); }
  100% { transform: scale(1);   opacity: 1; }
}

/* Number count up — handled in JS via useCountUp hook */

/* Apply staggered entry to page sections */
.page-section {
  animation: fadeUp var(--duration-slow) var(--ease-default) both;
}
.page-section:nth-child(1) { animation-delay: 0ms; }
.page-section:nth-child(2) { animation-delay: 80ms; }
.page-section:nth-child(3) { animation-delay: 160ms; }
.page-section:nth-child(4) { animation-delay: 240ms; }
.page-section:nth-child(5) { animation-delay: 320ms; }

/* Button press feedback — global */
button, [role="button"] {
  transition: transform var(--duration-fast) ease,
              opacity var(--duration-fast) ease;
}
button:active, [role="button"]:active {
  transform: scale(0.97);
  opacity: 0.85;
}

/* Skeleton loader */
.skeleton {
  background: linear-gradient(90deg,
    var(--color-bg-secondary) 25%,
    var(--color-bg-tertiary)  50%,
    var(--color-bg-secondary) 75%);
  background-size: 200% 100%;
  animation: shimmer 1.5s infinite;
  border-radius: var(--radius-md);
}
```
