**Why this prompt is structured this way:**

The audit-first instruction is the most important line in the entire prompt. Agent has a tendency to build fresh when it should be bridging. Forcing it to present findings before touching code means you catch duplications and conflicts before they get baked in.

The phase structure gives Agent a clear sequence so it doesn't jump around and create disconnected pieces.
