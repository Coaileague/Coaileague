Tenant Sandbox Policy


Each tenant_id is isolated; roles and data are scoped.


Root/Deputy can “impersonate” tenant users for audits.


AI automations run under system accounts tied to each tenant.


Human oversight only requires access to logs, tasks, and approvals.
