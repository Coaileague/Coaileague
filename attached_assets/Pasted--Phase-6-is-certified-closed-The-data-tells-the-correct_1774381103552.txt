**REJECTED — Assuming a try-catch is equivalent to a transaction.** A try-catch that catches a failure after the first write and attempts to clean up is not atomic. ```typescript
const result = await db.transaction(async (tx) => {

    .insert(primaryTable)
    .values(primaryData)
