The One Bug — WorkspaceId Resolution Returning Empty
The UI shows 0 thalamic records even though 35 exist in the DB. The root cause is clear from the trace:
The frontend query is passing workspaceId as a URL path segment instead of a query parameter. The middleware reads workspaceId from req.session.workspaceId as the fast path — but when the session's currentWorkspaceId is null or undefined, nothing resolves and the query returns empty.
