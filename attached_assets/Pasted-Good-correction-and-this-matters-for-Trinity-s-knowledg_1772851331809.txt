## TEXAS SECURITY LICENSE TERMINOLOGY — CORRECTED

In Texas the document issued by **Texas DPS Private Security Bureau** is officially called the **PERC card** (Personal Employment Registration Certificate) but in the industry and on the street it is commonly called a **Guard Card** — both terms refer to the same document.

**Texas License Levels:**
- **Level II** — Unarmed Security Officer (Guard Card, unarmed)
- **Level III** — Armed Security Officer (Guard Card, armed — requires firearms qualification)
- **Level IV** — Personal Protection Officer (Bodyguard level)
- **Level V and above** — Commissioned officer/management tiers

---

## HOW THIS DIFFERS STATE BY STATE

| State | What It's Called | Issuing Authority |
|-------|-----------------|-------------------|
| Texas | PERC Card / Guard Card (Level II/III/IV) | Texas DPS Private Security Bureau |
| California | Guard Card | BSIS — Bureau of Security and Investigative Services |
| Florida | Class D / Class G License | FDACS Division of Licensing |
| New York | Security Guard Registration | NY Secretary of State |
| Illinois | PERC Card (different from TX — Permanent Employee Registration Card) | Illinois State Police |
| Georgia | Security License | Georgia Board of Private Detective and Security Agencies |
| Arizona | Security Guard License | Arizona DPS |
