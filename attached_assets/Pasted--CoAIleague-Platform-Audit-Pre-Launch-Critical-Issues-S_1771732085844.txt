# CoAIleague Platform Audit — Pre-Launch Critical Issues Sweep

## 7-Step Systematic Audit Process

> **Purpose:** Eliminate race conditions, CRUD/persistence failures, database sync issues, redundant bandaids, hardcoded values, rendering bugs, and broken workflows across ALL services, routes, domains, and UI surfaces before going live.

> **How to use:** Run each grep/search command in your Replit shell. Every hit is a potential issue. Triage as CRITICAL / HIGH / MEDIUM / LOW.

---

## STEP 1: Race Conditions & Concurrency

Race conditions are the #1 silent killer in production. They work fine in dev with one user and explode under real traffic.

### 1.1 — Unguarded Async State Mutations

Search for state being set without awaiting the operation that produces the value:

```bash
# React setState called inside async without proper await guards
grep -rn "setState\|set[A-Z].*(" src/ --include="*.jsx" --include="*.tsx" --include="*.js" | grep -i "fetch\|axios\|api\|async"

# Missing await before database operations
grep -rn "\.create(\|\.update(\|\.delete(\|\.save(\|\.findOne(\|\.find(" src/ server/ --include="*.js" --include="*.ts" | grep -v "await"

# Fire-and-forget promises (no await, no .then, no .catch)
grep -rn "\.then(" src/ server/ --include="*.js" --include="*.ts" | grep -v "catch\|\.catch"
```

### 1.2 — Parallel Writes Without Locks or Transactions

```bash
# Multiple sequential writes that should be in a transaction
grep -rn "await.*\.save\(\)" server/ --include="*.js" -A2 | grep "await.*\.save\(\)"

# Optimistic updates without conflict resolution
grep -rn "findOneAndUpdate\|updateOne\|updateMany" server/ --include="*.js" | grep -v "version\|__v\|optimistic"

# Check for missing transaction wrappers on multi-step operations
grep -rn "async.*function\|async.*=>" server/ --include="*.js" -A20 | grep -c "\.save\(\)\|\.create\(\)\|\.update\("
```

### 1.3 — Stale Closures in React

```bash
# useEffect with missing dependencies (stale closure trap)
grep -rn "useEffect" src/ --include="*.jsx" --include="*.tsx" -A10 | grep "\[\]"

# setTimeout/setInterval inside components without cleanup
grep -rn "setTimeout\|setInterval" src/ --include="*.jsx" --include="*.tsx" | grep -v "clearTimeout\|clearInterval"

# Event listeners without cleanup
grep -rn "addEventListener" src/ --include="*.jsx" --include="*.tsx" | grep -v "removeEventListener"
```

### 1.4 — WebSocket / Real-time Race Conditions

```bash
# Socket events without deduplication
grep -rn "socket\.on\|io\.on" src/ server/ --include="*.js" --include="*.jsx" | grep -v "removeListener\|off("

# Multiple socket connections (no singleton pattern)
grep -rn "io(\|new WebSocket\|socketio\|socket\.connect" src/ --include="*.js" --include="*.jsx"

# Emitting before connection confirmed
grep -rn "socket\.emit" src/ --include="*.js" --include="*.jsx" -B5 | grep -v "connected\|isConnected\|socket\.on.*connect"
```

### 1.5 — AI Orchestration Race Conditions (Trinity/Claude/OpenAI)

```bash
# Parallel AI calls without proper fallback sequencing
grep -rn "Promise\.all\|Promise\.race" server/ --include="*.js" | grep -i "ai\|claude\|openai\|gemini\|trinity"

# Missing timeout on AI API calls
grep -rn "fetch\|axios" server/ --include="*.js" | grep -i "anthropic\|openai\|gemini\|googleapis" | grep -v "timeout"

# Fallback chain without proper error boundaries
grep -rn "catch" server/ --include="*.js" -A5 | grep -i "claude\|openai\|gemini"
```

## STEP 2: CRUD & Database Persistence

### 2.1 — Orphaned Records & Missing Cascades

```bash
# Delete operations without cascade/cleanup
grep -rn "\.deleteOne\|\.deleteMany\|\.findOneAndDelete\|\.remove(" server/ --include="*.js" -A5 | grep -v "cascade\|related\|clean\|associated"

# Foreign key references — check all ref fields have corresponding cleanup
grep -rn "ref:" server/models/ --include="*.js"

# Records created without required fields
grep -rn "new.*Model\|\.create(" server/ --include="*.js" -A10 | grep -v "required"
```

### 2.2 — Transaction Gaps

```bash
# Multi-document operations without transactions
grep -rn "async" server/ --include="*.js" -A30 | grep -P "(\.save\(\)|\.create\(|\.update\(|\.delete\().*\n.*(\.save\(\)|\.create\(|\.update\(|\.delete\()" 

# Manual check: search for any endpoint that modifies 2+ collections
grep -rn "Model\." server/routes/ --include="*.js" | awk -F: '{print $1}' | sort | uniq -c | sort -rn | head -20
```

### 2.3 — Missing Error Rollbacks

```bash
# Try/catch blocks that don't rollback on failure
grep -rn "try {" server/ --include="*.js" -A30 | grep "catch" -A5 | grep -v "rollback\|abort\|undo\|revert\|delete"

# API endpoints without proper error responses
grep -rn "router\.\(get\|post\|put\|patch\|delete\)" server/ --include="*.js" -A30 | grep "catch" -A3 | grep -v "res\.\(status\|json\|send\)"
```

### 2.4 — Data Sync Between Services

```bash
# Check for data being stored in multiple places (sync risk)
grep -rn "localStorage\|sessionStorage" src/ --include="*.js" --include="*.jsx" | grep -v "token\|theme"

# Redux/Zustand/Context state that mirrors DB without sync mechanism
grep -rn "createContext\|createStore\|useReducer" src/ --include="*.js" --include="*.jsx"

# Caching without invalidation
grep -rn "cache\|Cache\|cached\|memoize" server/ src/ --include="*.js" --include="*.jsx" | grep -v "invalidate\|clear\|expire\|ttl\|maxAge"
```

### 2.5 — Subscriber Org Data Isolation (CRITICAL for Multi-Tenant)

```bash
# Database queries WITHOUT organization/tenant filter
grep -rn "\.find(\|\.findOne(\|\.findMany(\|\.aggregate(" server/ --include="*.js" | grep -v "org\|organization\|tenant\|company\|subscriber"

# API routes missing org-scoped middleware
grep -rn "router\.\(get\|post\|put\|delete\)" server/routes/ --include="*.js" -B5 | grep -v "orgAuth\|tenantAuth\|orgMiddleware\|verifyOrg"

# AI token cost attribution — ensure all AI calls tag the org
grep -rn "anthropic\|openai\|gemini\|googleapis.*generate" server/ --include="*.js" | grep -v "orgId\|organizationId\|tenantId\|subscriberId"
```

## STEP 4: Redundancy & Bandaid Removal

### 4.1 — Duplicate Logic

```bash
# Find duplicate function names across files
grep -rn "function \|const.*=.*=>" src/ server/ --include="*.js" --include="*.jsx" | awk -F'[( =]' '{print $NF}' | sort | uniq -cd | sort -rn | head -30

# Duplicate API calls (same endpoint called from multiple components)
grep -rn "fetch\|axios\|api\." src/ --include="*.jsx" --include="*.tsx" | grep -oP '["'"'"'][^"'"'"']*["'"'"']' | sort | uniq -cd | sort -rn | head -20

# Duplicate utility functions
grep -rn "export.*function\|module\.exports" src/utils/ server/utils/ --include="*.js" 2>/dev/null
```

### 4.2 — TODO/FIXME/HACK/TEMP Tags (Bandaids)

```bash
# Find ALL temporary fixes that need permanent solutions
grep -rn "TODO\|FIXME\|HACK\|TEMP\|WORKAROUND\|BANDAID\|XXX\|KLUDGE" src/ server/ --include="*.js" --include="*.jsx" --include="*.ts" --include="*.tsx"

# Commented-out code blocks (dead code)
grep -rn "^\s*//.*function\|^\s*//.*const\|^\s*//.*return\|^\s*/\*" src/ server/ --include="*.js" --include="*.jsx" | head -50

# Console.log left in production code
grep -rn "console\.log\|console\.warn\|console\.error\|console\.debug" src/ server/ --include="*.js" --include="*.jsx" | grep -v "node_modules\|\.test\.\|\.spec\." | wc -l
```

### 4.3 — Overlapping Route Handlers

```bash
# Find duplicate route definitions
grep -rn "router\.\(get\|post\|put\|patch\|delete\)" server/routes/ --include="*.js" | awk -F'"' '{print $2}' | sort | uniq -cd

# Find duplicate Express middleware
grep -rn "app\.use\|router\.use" server/ --include="*.js" | sort

# Overlapping API patterns
grep -rn "app\.\(get\|post\|put\|delete\)" server/ --include="*.js" | sort
```

### 4.4 — Dead/Unreachable Code

```bash
# Exported but never imported
for f in $(grep -rln "export" src/ server/ --include="*.js" --include="*.jsx" 2>/dev/null); do
  exports=$(grep -oP "export\s+(default\s+)?(function|const|class)\s+\K\w+" "$f")
  for exp in $exports; do
    count=$(grep -rn "$exp" src/ server/ --include="*.js" --include="*.jsx" | grep -v "$f" | wc -l)
    if [ "$count" -eq 0 ]; then
      echo "UNUSED EXPORT: $exp in $f"
    fi
  done
done

# Unused npm dependencies
npx depcheck 2>/dev/null || echo "Install depcheck: npm install -g depcheck"
```

## AUTOMATED AUDIT SCRIPT

Save this as `audit.sh` in your Replit root and run with `bash audit.sh`:

```bash
#!/bin/bash
# CoAIleague Pre-Launch Audit Script
# Run from project root: bash audit.sh

RED='\033[0;31m'
YEL='\033[1;33m'
GRN='\033[0;32m'
NC='\033[0m'
TOTAL=0

check() {
  local label="$1"
  local severity="$2"
  shift 2
  local count

    TOTAL=$((TOTAL + count))
  else

  fi
```
