# COAILEAGUE - DEEP CRAWL SYSTEM AUDIT

## MISSION CRITICAL DIRECTIVE

This is a **ZERO-TOLERANCE** audit. Every API call, AI execution, database operation, workflow, button, link, and data flow must be traced end-to-end. The goal is to ensure:

1. **NO BROKEN API CALLS** - Every endpoint works and returns expected data
2. **NO BROKEN AI EXECUTIONS** - Trinity, Claude, Gemini, OpenAI all respond correctly
3. **NO SEMANTIC WORKFLOW GAPS** - Every automation triggers and completes
4. **NO CANONICAL ISSUES** - Consistent naming, routing, data structures
5. **NO DATA LEAKS** - Workspace isolation is absolute
6. **NO SESSION ISSUES** - User data persists correctly
7. **NO ORPHANED OPERATIONS** - Every action has a result

**METHODOLOGY:** Parallel sub-agents, systematic tracing, document EVERYTHING before fixing.
