# CoAIleague — Trinity FAB Final Fix + Home Dashboard Widget
## Stop Floating Over Content. Give Trinity Her Own Space.

---

## THE PROBLEM

Trinity's FAB has been colliding with the + button and covering page content on every page because it has no awareness of what else is on screen. This ends now.

## THE SOLUTION — Two Trinity Modes

**Mode 1 — Home Dashboard:** Trinity gets a dedicated widget card in the page layout. Not floating. Her own section with real estate between Earnings and the grid tiles.

**Mode 2 — All Other Pages:** Compact FAB, bottom LEFT, hides on scroll. Never competes with + button again.

```typescript
// Scroll hide/show logic — add to TrinityWidget.tsx
useEffect(() => {
  let lastScrollY = window.scrollY;
  let ticking = false;

  const handleScroll = () => {
    if (!ticking) {
      requestAnimationFrame(() => {
        const currentScrollY = window.scrollY;
        setFabVisible(currentScrollY < lastScrollY || currentScrollY < 80);
        lastScrollY = currentScrollY;
        ticking = false;
      });
      ticking = true;
    }
  };

  window.addEventListener('scroll', handleScroll, { passive: true });
  return () => window.removeEventListener('scroll', handleScroll);
}, []);
```
