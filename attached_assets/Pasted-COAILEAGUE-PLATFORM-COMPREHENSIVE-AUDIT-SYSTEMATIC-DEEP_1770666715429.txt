COAILEAGUE PLATFORM COMPREHENSIVE AUDIT - SYSTEMATIC DEEP ANALYSIS
OBJECTIVE: Conduct a complete platform audit across ALL systems, identifying gaps, redundancies, semantic issues, and ensuring Fortune 500-grade organization. Execute in phases to avoid context overflow.

PHASE 1: ARCHITECTURE & CODE ORGANIZATION AUDIT
1.1 Service Layer Analysis

Map ALL services in /server/services/ - identify redundant or duplicate services
Check for overlapping functionality between services (e.g., multiple QuickBooks services, multiple auth handlers)
Consolidate redundant webhooks, managers, handlers into single sources of truth
Document service dependencies and circular reference risks

1.2 Route & Endpoint Audit

Catalog ALL API endpoints across all route files
Verify RESTful naming conventions and semantic consistency
Check for duplicate endpoints doing similar things
Ensure all endpoints have proper authentication middleware
Verify all endpoints validate workspaceId and userId for org separation

1.3 Database Schema Review

Audit all tables for proper foreign key relationships
Verify workspaceId exists on ALL tenant-specific tables
Check for orphaned records potential
Verify indexes exist for frequently queried columns
Audit real-time sync mechanisms (WebSocket, polling)


PHASE 2: MULTI-TENANCY & SECURITY AUDIT
2.1 Organization Isolation Verification

EVERY database query must filter by workspaceId - find any that don't
Verify session-based org separation throughout all services
Check all CRUD operations use internally-assigned userId (not user-provided)
Audit that invited users are properly scoped to their organization
Verify cross-org data leakage is impossible

2.2 User Identity Chain

Trace user creation flow (self-signup vs invited vs created by admin)
Verify userId assignment is server-side only, never client-provided
Check all actions/executions are attributed to authenticated userId
Audit role-based access control (RBAC) enforcement on all endpoints

2.3 Authentication & Authorization

Review all requireAuth middleware usage
Check requireWorkspaceMembership is applied correctly
Verify API key authentication for external integrations
Audit session management and token refresh logic
