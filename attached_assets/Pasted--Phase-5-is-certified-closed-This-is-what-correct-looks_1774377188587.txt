Any `.toFixed(2)` found on an intermediate value is a Phase 2 regression that must be fixed.

**REJECTED ‚Äî Calculation logic inside route handlers.** Financial calculation belongs in `financialCalculator.ts`. Any arithmetic found directly inside a route handler or service that is not a call to `FinancialCalculator` is inline calculation that must be moved.

**REJECTED ‚Äî Database values converted before calculation.** A pay rate read from the database as a string must remain a string until it enters a `FinancialCalculator` function. Any `parseFloat()` or `Number()` applied to a database value before passing it to a calculation is a precision violation.
